# C&W Korea WPR 가이드 앱 — 리디자인 제안서
> **벤토 디자인 × 스마트폰 UI** 리디자인 안

---

## 1. 디자인 방향 개요

### 핵심 변경 사항

| 항목 | 현재 (As-Is) | 변경 후 (To-Be) |
|---|---|---|
| 배경 | 웹툰 애니메이션 영상 재생 | **숨김 처리** (제거) |
| 전체 프레임 | 데스크톱 모니터 오버레이 | **아이폰 목업 프레임** |
| 기본 화면 | 폴더 그리드 (기능별) | **메신저형 직원 목록** |
| 직원 클릭 시 | 없음 | **우측 슬라이드 패널** (현행 동일) |
| 탭 전환 | 기능별 / 담당자별 탭 | **하단 탭바** (메신저 / 업무목록) |
| 업무목록 뷰 | 4열 폴더 그리드 | **벤토 그리드 카드** |
| 디자인 언어 | 웹툰/만화 스타일 | **iOS + 벤토 그리드** |


**스타일 속성:**
- 카드 배경: `#FFFFFF`
- 카드 radius: `16px`
- 아바타 크기: `48×48px`, 원형, 색상은 직원별 컬러
  - 권희원: `#8B5CF6` (보라)
  - 김경만: `#3B82F6` (파랑)
  - 김지성: `#F59E0B` (골든)
  - 송진영: `#EC4899` (핑크)

- 카드 하단에 `1px` 구분선 (`#F2F2F7`)
- 카드 탭 시 `scale(0.97)` 스프링 애니메이션

### 3-3. 직원 클릭 → 우측 슬라이드 패널

직원 카드 클릭 시 현재 앱과 동일한 방식으로 우측에서 슬라이드 패널이 등장합니다.

```
┌──────────────┬────────────────┐
│  직원 목록   │ ← 뒤로         │
│  (흐림 처리) │  김경만 과장   │
│              │  Noel Kim      │
│              │─────────────── │
│              │ 담당 업무 목록  │
│              │                │
│              │ 🖊️ 인감 날인   │
│              │ 🖥️ Solstice    │
│              │ 📹 Canteen Rm  │
│              │ 🖨️ Printix     │
│              │                │
│              │ [업무 클릭 →   │
│              │  상세 슬라이드] │
└──────────────┴────────────────┘
```

- 패널 너비: 아이폰 화면 `100%` (풀스크린)
- 슬라이드 방향: 오른쪽에서 왼쪽으로 (`translateX(100%) → 0`)
- 트랜지션: `300ms cubic-bezier(0.25, 0.46, 0.45, 0.94)`
- 뒤에 있는 직원 목록: `backdrop-filter: blur(4px)` + `opacity: 0.5`
- 업무 클릭 시 동일한 방식으로 세부 슬라이드 패널 재등장

### 4-2. 벤토 카드 크기 분류

| 카드 크기 | 할당 업무 | 설명 |
|---|---|---|
| **Small** (1×1) | 공문번호, 명함, 계약서, 화환, 락커, 인감, 법인차량, Solstice, Canteen, Printix | 기본 정사각형 |
| **Wide** (2×1) | 국내출장, 해외출장, Supplier | 가로로 넓게 강조 |

> 출장 관련 업무(국내/해외)와 Supplier는 업무 비중이 높아 와이드 카드로 배치

### 4-3. 벤토 카드 디자인 스타일

```
╔══════════════════════════════╗
║  📄                         ║
║                              ║
║  공문번호 발급               ║
║  행정                        ║  ← 배지
╚══════════════════════════════╝
```

**카드 스타일 속성:**
- 배경: 각 업무별 개성 있는 파스텔 그라디언트
  - 행정 계열: `#FFF9E6 → #FDEFC8` (웜 옐로우)
  - 출장 계열: `#E3F2FD → #BBDEFB` (스카이 블루)
  - IT 계열: `#E8EAF6 → #C5CAE9` (인디고)
  - 시설 계열: `#E0F2F1 → #B2DFDB` (민트)
- 카드 radius: `20px`
- 이모지 크기: Small `28px` / Wide `36px`
- 업무명 폰트: `Noto Sans KR`, `font-weight: 600`
- 배지: `6px radius`, 카테고리별 컬러 도트
- Hover/탭 시 `scale(0.95)` + `box-shadow` 강조
- 클릭 시 현재와 동일한 **우측 슬라이드 패널** 오픈

## 6. 슬라이드 패널 상세 (공통)

직원 선택 또는 업무 카드 클릭 시 공통으로 사용하는 패널입니다.

```
┌─────────────────────────────┐
│ ← 뒤로     공문번호 발급    │  ← 상단 바
│            행정 · Gina Kim  │
├─────────────────────────────┤
│  [담당자 카드]               │
│  🟡 김지성 대리              │
│     gina.kim@cushwake.com   │
├─────────────────────────────┤
│  📋 진행 플로우              │
│  ●──●──●──●──●             │
│  작성 요청 회신 기재 날인     │
├─────────────────────────────┤
│  Step 1. 공문 작성           │
│  [내용]                     │
│                             │
│  Step 2. 메일 발송           │
│  ┌─────────────────────┐   │
│  │ 📧 이메일 템플릿     │   │
│  │ To: 부서장          │   │
│  │ CC: Gina Kim        │   │
│  │ Subject: [승인요청]..│   │
│  │ [본문 내용...]       │   │
│  │        [복사 버튼]   │   │
│  └─────────────────────┘   │
│                             │
└─────────────────────────────┘
```

- 패널 배경: `#FFFFFF`
- 상단 바: `backdrop-filter: blur(20px)`, iOS 스타일 sticky
- 담당자 카드: 직원 아바타 + 연락처 클릭 시 `mailto:` 링크
- 플로우 표시: 수평 도트-라인 타임라인
- 이메일 템플릿 블록: `#F2F2F7` 배경의 코드 블록 스타일
- 각 필드(To/CC/Subject/Body) 개별 복사 버튼 유지
- 스크롤: 패널 내부 독립적 스크롤
